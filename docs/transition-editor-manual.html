Refinement

Where sequence becomes arrangement.

The Transition Editor is for shaping relationships between placements, not for pretending to be a live booth. This is where overlap windows, transition style, Start Here, and snapshot discipline matter.

  • Transition Style chooses the handoff character.
  • Overlap changes how much time two placements share.
  • Start Here lets you audition from the current transition forward.
  • Save Before Snapshot preserves a clean rollback point before bolder changes.

Segment Pool Lane

Segment Pool is the app's answer to more creative club arrangement. A source track can contribute multiple phrase-safe slices at different points in the set if their local BPM, key, texture, and transition context make sense.

  • Drag-reorder slices when you want a different re-entry cadence.
  • Use snapshots before aggressive style changes or recall moves.
  • Watch compatibility, vocal risk, kick-safe score, and bassline conflict before forcing a dramatic overlap.
