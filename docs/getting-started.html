Session 01

From first import to finished export.

The shortest reliable path through modDJs is import, analyze, build, preview, export. This page is the practical first-session route.

  1. Open the app and create a new project.
  2. Use Files or Folder to import audio.
  3. Wait for analysis badges to populate BPM, key, phrasing, and energy.
  4. Choose a target duration and preset in Set Builder.
  5. Press Build to generate the first arrangement.
  6. Open Timeline or Transition Editor to refine the result.
  7. Open Export Studio, choose format and mastering settings, then print the set.

First build checklist

  • Use tracks that already share a broad style family for the first pass.
  • If the set feels too linear, switch section mix mode to section-driven or club-edit.
  • For tighter club pacing, lower preserve-full-tracks and raise section creativity.
  • If files move on disk, use relink before editing transitions.

Top bar actions

AnalyzeRuns or reruns analysis across the current track pool.After import, after relink, or after manual marker changes.

PreviewStarts timeline-aware playback or transition audition from the current context.Use before export to verify pacing and overlap quality.

What a good first project looks like

  • 6 to 12 tracks with roughly compatible groove and emotional direction.
  • Most tracks analyzed successfully with visible key/BPM confidence.
  • A target duration close to what the pool can realistically hold.
  • Transition smoothness around the middle before pushing into more aggressive edits.
  • Export profile left on balanced unless you specifically need low-impact rendering.
