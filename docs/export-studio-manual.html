Final Print

Export is treated as a studio phase, not a blind bounce.

Export Studio exposes render phases, ETA, subtask state, mastering metrics, compare views, and output paths. It is meant to tell you exactly what is happening when the mix becomes a deliverable file.

  • WAV, MP3, FLAC export targets
  • Master chain and loudness target controls
  • Reveal output path and export history
  • Before/after compare metrics and preview clips

Export controls reference

ControlPurposeOperator note
FormatSelect WAV, MP3, or FLAC output.Use WAV for archive/master, MP3 for fast delivery.
ProfileChoose low-impact, balanced, or high-quality render profile.Use low-impact if desktop load matters more than speed parity.
Loudness TargetSets target LUFS for final pass.Use more conservative values if club systems already add pressure.
RevealOpens Finder or Explorer at the saved output path.Use after every successful print to verify the actual artifact.
A/B CompareLets you hear and measure before vs after mastering.Useful for checking whether the chain solved a problem or created one.

Release discipline

  • Do not publish a release unless Apple Silicon smoke, Windows smoke, and mac Intel compatibility smoke are green.
  • Keep 10, 30, and 60 minute native export benchmarks inside the release gate.
  • Use low disk preflight and relink smoke as blockers, not optional checks.
